## Ход выполнения работы

### 1. Создание проекта и настройка окружения

Был создан новый проект в Android Studio по шаблону **Empty Activity**.  
Проект был настроен на использование **ComponentActivity**.  
В качестве основного способа отображения контента вместо XML-разметки использован собственный класс `DrawView`, наследуемый от `View`.

### 2. Реализация задания 2

- Название приложения изменено через метод `setTitle()`.
- На экране отображается информация об авторе и дата рождения с помощью `canvas.drawText()`.

### 3. Реализация задания 3 (вариант 1)

Нарисован **зелёный прямоугольник**, у которого высота в 2 раза больше ширины.

### 4. Реализация задания 4 (вариант 1)

Нарисован **дом с дверью и окном** с использованием следующих примитивов:
- `drawRect()` — стены и дверь
- `drawPath()` — крыша (треугольник)
- `drawCircle()` — дверная ручка
- `drawRect()` + `drawLine()` — окно с рамкой и перекладинами
